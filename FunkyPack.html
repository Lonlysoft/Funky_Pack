Controls

touchscreen keyboard equivalent gamepad equivalent what it does in the game world?
WASD left Analog or d-pad moves character
spacebar A makes the character jump
e B makes the character interact with things
v L+Y makes the character vore objects or people
mouse click / q X use current tool
" Y makes the character hold an object
R RT makes the character run
shift / c LT makes the character crouch
enter pause/start opens character inventory
scroll lock options/select pauses game without menu or music
ctrl right analog button open quick settings
